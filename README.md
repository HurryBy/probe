# IP探针

生成探针，发送探针获取他人信息。

### 测试网站

网页版测试网站: https://abcd.humorously.tk

> 测试网站搭建于 Amazon US,加载较慢

### 如何安装

下载release中的最新版本，解压里面的文件至网站根目录

访问/install.html填入数据库信息进行安装
### 功能列表

- [x] 可获取GPS,IP,摄像头,浏览器信息
- [x] 支持在线查询
- [x] 可自定义程序高
- [x] 免费开源

### 介绍

生成探针，发送探针获取他人信息。
当时主要是想着网络上没有什么好用的探针，有的也是年久失修了
然后就突发奇想自己开发一个玩玩，也开发了挺久，希望支持一下！
